# Stock-Analyser

A Streamlit-based stock price analysis and trend prediction dashboard using Yahoo Finance data, technical indicators, LSTM forecasting, and PDF export.

## Overview
This project fetches historical stock data, calculates technical indicators, visualizes trends, and predicts future prices using an LSTM model. It also provides CSV export and PDF report generation.

## Features
- Real-time stock data from Yahoo Finance
- Moving averages
- RSI, MACD, and Bollinger Bands
- Trend signal detection
- LSTM-based prediction
- Future forecast
- CSV and PDF export

## Files
- `app.py` - Main Streamlit dashboard
- `model.py` - Data fetching, indicators, and prediction logic
- `charts.py` - Chart generation
- `report.py` - PDF report generator
- `requirements.txt` - Python dependencies

## Installation
```bash
pip install -r requirements.txt
```

## Run Locally
```bash
streamlit run app.py
```
